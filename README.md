## Overview

Celery is a Windows desktop utility for Roblox Lua workflows. Its core functions include Lua injection and rapid script execution, while the interface is intended to remain straightforward for Windows 10/11 users. A built-in hub provides convenient access to a large collection of scripts without requiring a key-based startup process.

Rather than filling the application with unnecessary controls, Celery concentrates on everyday execution tasks. Persistent queuing, debugging support, and automatic updates are included to make repeated scripting sessions easier to manage.

## Feature Highlights

- Start Roblox scripts through a simple one-click execution flow
- Browse a built-in library with more than 500 available scripts
- Save queue data and execution history using SQLite-backed storage
- Keep the application current through its automatic update engine
- Use the interface in multiple supported languages
- Execute several scripts consecutively with batch mode
- Examine script behavior with the integrated debugger
- Begin without a key on supported systems

Celery is intended for Windows only, with Windows 10 and Windows 11 as the target systems.
